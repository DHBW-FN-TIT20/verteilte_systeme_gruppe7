## :computer: Contributing

### :gear: compile via script
grant execute permission to the file:
```shell
chmod +x ./scripts/compile.sh
```
compile all:
```shell
./scripts/compile.sh
```
add parameters to compile individual files:
```
-p publisher.cpp
-b broker.cpp
-s subscriber.cpp
```
